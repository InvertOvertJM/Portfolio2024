Game Picker

Reflection

The Game Picker was a project during Junior Year where we had to make a program with multiple outputs from the users' input. Using applab, I created an app that took your age, price, and genre of game that you want. I then selected multiple games from my steam library which I gave a short synopsis about for each recommendation. This project was a good test of skills during the middle of the Year in order to cement my programming knowledge in preperation for the CTE Expo and upcoming PT Project.

Website for Change

Reflection

The Website for Change project was the second final project of Junior year where where we had to identify some sort of issue and create a website about it, informing others and providing possible solutions about it. Rather than picking a more serious topic I decided to do something about video games as I am very passionate about them. By working on a project I was passionate about, I was able to create a more stylistic website utilizing Bootstrap, an external stylesheet for CSS.

Performance Task

Reflection

The Performance Task was a major project in Junior year where we had to meet a set of requirements in our code in order to qualify for the AP exam. For this, I created a game that satisfied all conditions, being inspired by card games I play and rock paper scissors. This project was a great way of testing my programming knowledge as I had to create a program in mind that would be questioned in the AP exam. The hardest thing about the project for me was creating the clashing system, where the moves of the cpu and player would be compared and a random number rolled, taking the higher number move and applying the effect.

Video Game Zoo Website

Reflection

The Video Game Zoo Website was the final project for Sophomore Year, testing our ability to design websites in HTML and CSS by creating a Zoo website. Utilizing Bootstrap, an external stylesheet for CSS, I was able to create a website based on one of my favorite games, Lobotomy Corporation. Lobotomy Corporation was a game I thought was suitable for the topic and I also wanted to do something unique, rather than basing the website off of real animals. The game takes place in a dystopian city where corporations run districts and the government barely interacts with them, making a very rough world. Lobotomy Corporation is a company whos goal is to interact with Abnormalities, objects and creatures that have anomalous properties, creating energy for the city and selling it to other corporations.

Video Game Informational App

Reflection

The app based on the game "Faith" by Airdorf was one of the first major projects of Junior year and tested our ability to design and create navigation through code.org's applab. Utilizing the applab javascript me and a friend collaborated to create the app, adding comments to document everything in our code, while also coming up with a design that suited the game. While I handled many of the navigation aspects, my friend came up with the design using the red and black color scheme that is present in the games' promotional material. Being able to collaborate on a project allowed me to work on my teambuilding and communication skills as many things had to be specific in order to work.
